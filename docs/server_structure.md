## Server Structure

The files that are uploaded to the server follow this structure:

```
index.html
packages/
  |- [package_name]/
  |    |- cover.png
  |    |- audio.mp3
  |    |- transcript.json
  |    |- meta.json
  |    |- [other files included in the package]...
  |- [another package]/...
  |...
```
